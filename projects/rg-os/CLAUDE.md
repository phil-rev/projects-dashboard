## The Factory Framework (Quick Reference)
Every revenue stream has 5 factories:
1. **Attention → Leads** (form fills, downloads, registrations)
2. **Leads → Opportunities** (calls scheduled, meetings booked)
3. **Opportunities → Clients** (proposals, deal close)
4. **New Clients → Happy Clients** (onboarding, first success)
5. **Happy Clients → Ascended Clients** (upsell, expansion, referrals)

Each factory has: BUILD (HubSpot setup), ADOPT (team training), INTELLIGENCE (dashboards/KPIs)
