# Project Skills

Add custom skills for this project here. Skills are markdown files with commands or instructions that Claude can use.

## Example Skill Structure

```markdown
# Skill Name

Description of what this skill does.

## Usage
When to use this skill.

## Commands
```bash
# Actual commands to run
```
```

## Naming Convention
- Use kebab-case: `example-skill.md`
- Describe the action clearly

## Tips
- Keep skills focused on one task
- Reference your project's CLAUDE.md for context
- Test skills before documenting them
