# Synonyms Flashcards

## Goal
Interactive flashcard application for studying DSRP (Differentiated Semantic Relationship Pattern) synonyms. Build vocabulary through spaced repetition and interactive learning.

## What is DSRP?
DSRP is a learning methodology focused on understanding semantic relationships and nuances between similar words. This tool helps learners:
- Master synonym distinctions
- Build active vocabulary
- Learn through repetition and practice
- Track progress

## Features
- Interactive flashcard interface
- Spaced repetition algorithm
- Progress tracking
- Multiple learning modes
- Customizable decks
