# Synonyms Flashcards - Claude Instructions

## Project Overview
Educational web application for learning DSRP synonyms. Users study vocabulary through interactive flashcards with spaced repetition.

## What is DSRP?
Differentiated Semantic Relationship Pattern—a learning methodology focused on understanding relationships and nuances between similar words.

## Core Concepts
- **Flashcards**: Interactive cards showing words/definitions
- **Spaced Repetition**: Algorithm that optimizes when to show cards
- **Progress Tracking**: Users see their learning progress
- **Decks**: Collections of related synonym pairs
